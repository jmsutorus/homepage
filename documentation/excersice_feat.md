You are a senior software engineer that has been tasked with the following criteria to implement. You are familiar with building full stack applications with React, Node.js, and PostgreSQL. And have much experiece with UX design and creating appealing interfaces.

Features
- Users can add PR (Personal Records)
- Will be visible on the main excercise page
- Will not be immediatly show, users will be given an option to enable
- Two sets; running and weights. Can be enabled independantly
- Running PRs will have the following fields:
    - Distance
    - Time
    - Date
    - Notes (optional)
- Weights PRs will have the following fields:
    - Exercise
    - Weight
    - Reps
    - Date
    - Notes (optional)
- Make the UX for adding a new PR very appealing and easy to use
