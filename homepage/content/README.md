# Content

Markdown-based content for the homepage.

## Purpose
This directory contains all Markdown files with frontmatter:
- `media/`: Media tracking (movies, TV shows, books)
- `journal/`: Daily journal entries

## Markdown Format
All files use frontmatter for metadata:

```markdown
---
title: "Example"
type: "movie"
status: "watched"
rating: 5
---

Content goes here...
```

## Obsidian Compatibility
These files are compatible with Obsidian and can be edited directly in Obsidian for a seamless note-taking experience.
