# Database Utilities

SQLite database utilities and CRUD operations.

## Purpose
This directory contains:
- Database connection singleton (`index.ts`)
- Database schema (`schema.sql`)
- Seed data script (`seed.ts`)
- CRUD operations for mood entries (`mood.ts`)
- CRUD operations for tasks (`tasks.ts`)
- API cache operations (`cache.ts`)

## Database Tables
- `mood_entries`: Mood tracking data
- `tasks`: Task manager data
- `api_cache`: Cached API responses
