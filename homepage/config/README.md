# Configuration

JSON configuration files for the homepage.

## Purpose
This directory contains:
- `links.json`: Quick links configuration
- `homeassistant.json`: Home Assistant entity configuration

## Example: links.json
```json
{
  "categories": [
    {
      "name": "Development",
      "links": [
        { "title": "GitHub", "url": "https://github.com", "icon": "github" }
      ]
    }
  ]
}
```
